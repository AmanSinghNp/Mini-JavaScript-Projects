# Trie (Prefix Tree) Visualization

An interactive visualization of a **Trie** for storing strings.

## Features

-   **Insert**: Add words character by character.
-   **Search**: Check if a word exists in the trie.
-   **Prefix Structure**: Visualizes shared prefixes among words.

## How to Run

1.  Open `index.html` in your web browser.
2.  Insert words (e.g., "cat", "car") to see them share nodes.
3.  Search for words to highlight the traversal path.
