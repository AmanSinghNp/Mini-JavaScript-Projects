# Disjoint Set (Union-Find) Visualization

An interactive visualization of the **Disjoint Set Union (DSU)** data structure.

## Features

-   **Union**: Merge two sets (Union by Rank).
-   **Find**: Locate the representative of a set (Path Compression).
-   **Visuals**: Represents sets as trees that merge and flatten over time.

## How to Run

1.  Open `index.html` in your web browser.
2.  Create a set of elements.
3.  Perform **Union** and **Find** operations to see the structure evolve.
