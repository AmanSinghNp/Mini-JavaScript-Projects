# Hash Table Visualization

An interactive visualization of a **Hash Table** using Chaining for collision resolution.

## Features

-   **Put**: Insert or update key-value pairs.
-   **Get**: Retrieve values by key.
-   **Remove**: Delete entries.
-   **Visuals**: Shows the internal array (buckets) and linked lists (chains) formed by collisions.

## Logic
-   **Hash Function**: `Key % 10`
-   **Collision Handling**: Chaining (Linked List)

## How to Run

1.  Open `index.html` in your web browser.
2.  Enter a numeric **Key** and text **Value**.
3.  Click **Put** to insert.
