# Recursion Tree Visualization

An interactive visualization of recursive function calls, demonstrated with **Fibonacci Numbers**.

## Features

-   **Call Stack**: Visualizes the tree of recursive calls.
-   **Execution Flow**: Shows the order of calls (downward) and return values (upward).
-   **Visuals**: Nodes light up as they are active, waiting, or returned.

## How to Run

1.  Open `index.html` in your web browser.
2.  Enter a small number (e.g., 5).
3.  Click **Run** to watch the recursion unfold.
