# Queue Visualization

A simple, interactive visualization of the **Queue** data structure (FIFO - First In, First Out).

## Features

- **Enqueue**: Add elements to the rear of the queue.
- **Dequeue**: Remove elements from the front of the queue.
- **Peek**: View the front element without removing it.
- **Clear**: Remove all elements.
- **Visuals**: Animated entry and exit, capacity limits (Max 7), and responsive design.

## How to Run

1. Simply open `index.html` in your web browser.
2. Use the input field to type a value (max 5 chars).
3. Click buttons or press Enter to interact.
