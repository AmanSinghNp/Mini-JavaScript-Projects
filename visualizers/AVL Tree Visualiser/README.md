# Binary Search Tree (BST) Visualization

An interactive visualization of a **Binary Search Tree**.

## Features

-   **Insert**: Add nodes maintaining BST property (Left < Node < Right).
-   **Search**: Visualize the traversal path to find a node.
-   **Clear**: Reset the tree.
-   **Visuals**: Dynamic tree layout with connection lines and traversal animations.

## How to Run

1.  Open `index.html` in your web browser.
2.  Use the inputs to build and search the tree.
