# Dynamic Programming Visualization

An interactive visualization of the **0/1 Knapsack Problem** using Dynamic Programming.

## Features

-   **Table Construction**: Watch the DP table being filled cell by cell.
-   **Dependencies**: Highlights previous values used to calculate the current cell (Include vs Exclude item).
-   **Result**: Shows the maximum value achievable for the given capacity.

## How to Run

1.  Open `index.html` in your web browser.
2.  Add items (Weight, Value) and set Knapsack Capacity.
3.  Click **Run** to animate the table filling process.
