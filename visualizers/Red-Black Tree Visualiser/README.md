# Red-Black Tree Visualization

An interactive visualization of a **Red-Black Tree**, a self-balancing binary search tree.

## Features

-   **Insert**: Add nodes while maintaining Red-Black properties (coloring, rotations).
-   **Search**: Find nodes with path highlighting.
-   **Visuals**: Distinct red and black nodes, showing the balancing process.

## How to Run

1.  Open `index.html` in your web browser.
2.  Insert numbers to see the tree grow and balance itself.
