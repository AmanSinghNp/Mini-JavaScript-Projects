# Deque (Double-Ended Queue) Visualization

An interactive visualization of a **Deque** (Double-Ended Queue).

## Features

-   **Push Front/Rear**: Add elements to either end of the queue.
-   **Pop Front/Rear**: Remove elements from either end.
-   **Visual Layout**: Horizontal arrangement showing front and rear clearly.

## How to Run

1.  Open `index.html` in your web browser.
2.  Enter a value and use the Front/Rear buttons to modify the deque.
