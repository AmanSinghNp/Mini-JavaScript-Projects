# Graph Traversal Visualization

An interactive visualization of **Graph Traversal Algorithms** (BFS and DFS).

## Features

-   **Build Graph**: Add vertices and edges to create custom graphs.
-   **BFS**: Visualizes Breadth-First Search (queue-based).
-   **DFS**: Visualizes Depth-First Search (stack-based).
-   **Visuals**: Highlights current node, neighbors, and visited path.

## How to Run

1.  Open `index.html` in your web browser.
2.  Add nodes and edges manually or click **Seed Demo Graph**.
3.  Enter a start node and click **Run BFS** or **Run DFS**.
