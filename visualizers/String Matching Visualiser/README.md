# String Matching Visualization (KMP)

An interactive visualization of the **Knuth-Morris-Pratt (KMP)** algorithm.

## Features

-   **LPS Table**: Visualizes the construction of the Longest Prefix Suffix array.
-   **Search**: Animates the scanning of the text and pattern.
-   **Optimization**: Shows how the pattern shifts intelligently on mismatch.

## How to Run

1.  Open `index.html` in your web browser.
2.  Enter a Text string and a Pattern string.
3.  Click **Run** to start the visualization.
