# Segment Tree Visualization

An interactive visualization of a **Segment Tree** for range queries.

## Features

-   **Build**: Create a tree from a custom array.
-   **Query Range**: Calculate the sum of a specific range `[L, R]`.
-   **Update**: Modify a value in the array and see the changes propagate up the tree.

## How to Run

1.  Open `index.html` in your web browser.
2.  Input an array to build the tree.
3.  Use the controls to query ranges or update values.
