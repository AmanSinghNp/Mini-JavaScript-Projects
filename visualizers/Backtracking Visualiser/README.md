# Backtracking Visualization (N-Queens)

An interactive visualization of the **N-Queens Problem** using Backtracking.

## Features

-   **Solve**: Places queens row by row, checking for conflicts.
-   **Backtrack**: Visualizes the algorithm retreating when no valid move exists.
-   **Interactive**: Adjust board size (N) and animation speed.

## How to Run

1.  Open `index.html` in your web browser.
2.  Set N (4-8) and speed.
3.  Click **Solve** to watch the algorithm find a solution.
