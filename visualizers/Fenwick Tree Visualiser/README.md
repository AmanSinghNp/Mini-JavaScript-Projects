# Fenwick Tree (Binary Indexed Tree) Visualization

An interactive visualization of a **Fenwick Tree** for efficient prefix sums and updates.

## Features

-   **Build**: Initialize from an array.
-   **Query**: Compute prefix sums `Sum(1...i)` efficiently.
-   **Update**: Add a value to an index and update related nodes.
-   **Visuals**: Shows the implicit tree structure and binary index relationships.

## How to Run

1.  Open `index.html` in your web browser.
2.  Build the tree and perform queries or updates.
