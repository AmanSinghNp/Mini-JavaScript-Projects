# Priority Queue Visualization

An interactive visualization of a **Priority Queue**.

## Features

-   **Enqueue**: Add items with a specific priority value.
-   **Dequeue**: Remove the item with the highest priority (lowest number).
-   **Sorted View**: Automatically maintains items in order of priority.

## How to Run

1.  Open `index.html` in your web browser.
2.  Enter an item name and its priority (integer).
3.  Click Enqueue to add it to the correct position.
