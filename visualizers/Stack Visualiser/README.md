# Stack Visualization

An interactive visualization of a **Stack** data structure.

## Features

-   **Push**: Add an element to the top of the stack.
-   **Pop**: Remove the element from the top of the stack.
-   **Peek**: View the top element without removing it.
-   **LIFO Principle**: Demonstrates Last-In-First-Out behavior.
-   **Visual Feedback**: Animations for push and pop operations.

## How to Run

1.  Open `index.html` in your web browser.
2.  Use the input field to push values.
3.  Click buttons to perform operations.
