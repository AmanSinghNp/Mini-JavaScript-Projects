# B-Tree Visualization

An interactive visualization of a **B-Tree** (Degree 3 / 2-3 Tree variant).

## Features

-   **Insert**: Add keys and watch nodes split when full.
-   **Search**: Traverse the multi-way tree to find keys.
-   **Structure**: Demonstrates how B-Trees grow upwards and keep height low.

## How to Run

1.  Open `index.html` in your web browser.
2.  Insert numbers to observe node splitting and key distribution.
