# Linked List Visualization

A simple, interactive visualization of a **Singly Linked List**.

## Features

-   **Append**: Add nodes to the end.
-   **Prepend**: Add nodes to the start.
-   **Remove**: Find and remove specific values.
-   **Clear**: Reset the list.
-   **Visuals**: Nodes connected by arrows, animated traversal for removal, and capacity limits.

## How to Run

1.  Open `index.html` in your web browser.
2.  Use the inputs to add or remove values.
